# 📌 Problem Statement

In many real-world applications such as navigation systems, robotics, and game AI, finding an optimal path from a source to a destination in a dynamic environment is a critical challenge. Traditional pathfinding algorithms like Breadth-First Search (BFS), Uniform Cost Search (UCS), and A* are widely used, but their performance varies depending on environmental complexity, obstacle density, and dynamic changes.

This project aims to design and implement an intelligent pathfinding system that can efficiently navigate a grid-based environment while handling both static and dynamic obstacles. The system should be capable of real-time decision-making, adaptive replanning, and comparative evaluation of different algorithms.

---

# 🎯 Objectives

- To implement classical AI search algorithms (BFS, UCS, A*)
- To design a grid-based environment with configurable obstacles
- To simulate dynamic obstacles that change positions over time
- To enable real-time path replanning when the environment changes
- To compare algorithm performance based on:
  - Execution time
  - Nodes expanded
  - Path cost
- To visualize the agent’s movement and decision-making process

---

# 🌍 Scope

- The system operates in a 2D grid environment
- Supports both static and dynamic obstacles
- Allows experimentation with different map sizes and densities
- Provides graphical visualization and interactive UI
- Can be extended to real-world applications such as:
  - Autonomous navigation
  - Robotics path planning
  - Traffic routing systems

---

# 🚧 Challenges Addressed

- Handling dynamic changes in the environment
- Efficient path recalculation (replanning)
- Balancing optimality vs computational cost
- Visualizing algorithm behavior in real-time

---

# 💡 Expected Outcome

The project will demonstrate how different pathfinding algorithms perform under varying conditions and highlight the importance of heuristics and adaptive strategies in dynamic environments. It will also provide an intuitive interface to observe and analyze agent behavior step-by-step.

---

# 🧠 Significance

This project bridges theoretical AI concepts with practical implementation, showcasing how intelligent systems can adapt to changing environments and make efficient decisions in real time.
